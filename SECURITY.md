# Security Policy

## Supported Versions

| Version                    | Supported |
|----------------------------|-----------|
| 0.1.0 | Yes       |

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Use GitHub Security Advisories to report privately:
<https://github.com/ByronWilliamsCPA/Unify/security/advisories/new>

Or email: byronawilliams@gmail.com

## Response Timeline

We are committed to responding to valid security reports within **14 days** of receipt.

- Acknowledgement: within 48 hours
- Initial assessment: within 5 business days
- Status update: within 14 days
- Resolution target: 30 days for critical, 90 days for others
